Wiseance

A web app all about

saving and managing money

Features

Login system allows users to keep their information seperate from others

CAPTCHA integration makes sure no bots are using the site

Dynamic Amazon Wishlist

Dynamically finds Amazon products from links provided by the user

Keeps track of items with an easy to use wishlist that uses AJAX to dynamically manage list items

Alerts users to sales

Robust Budgeter

Easy to use setup with simple catagories

Data visulisation with charts

Allows user to put in expenditures and alerts them if they go over budget
